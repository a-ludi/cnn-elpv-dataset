## Install

This project uses [`pipenv`](https://github.com/pypa/pipenv) to manage its
dependencies. You may install it via `pip install pipenv`.

```
git clone --recurse-submodules https://github.com/a-ludi/cnn-elpv-dataset.git
pipenv install
```
